Completing A Strang Of DNA

OVERVIEW

This program reads a DNA sequence from a text file, creates complementary pairs,  and revers the sequence.
It is a solution to the "Completing A Strang Of DNA" Rosalind problem (ID: REVC). The tool is simple, efficient, and ideal for practicing file handling and string processing in Python.

FEATURES

Reads DNA sequence from a file (rosalind_revc.txt)
Automatically converts input to uppercase
Creates complementary pairs for the DNA
Reverses the DNA sequence
Fast and memory-efficient — works well with long sequences
Clean, well-commented code with proper functions and docstrings


EXAMPLE

Input (rosalind_revc.txt):TCCGTAAGACTAGC
Output:UCCGUAAGACUAGC

HOW IT WORKS

The program reads the DNA sequence from the file.
It cleans the input (removes whitespace and converts to uppercase).
It changes all T laters in file to U and puts everything in the (rna) sequence.
Finally, it prints the RNA sequence.
